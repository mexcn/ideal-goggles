# 💰 Finance Telegram Bot

Telegram бот для управления личными финансами с учетом расходов, генерацией отчетов, планированием бюджета и поддержкой нескольких валют.

## 🎯 Возможности

### ✅ Основной функционал

- **💰 Учет расходов**
  - Добавление расходов с категорией и описанием
  - Быстрое добавление через текстовые сообщения
  - Поддержка разных валют
  - Автоматическая конвертация валют

- **📊 Отчеты и статистика**
  - Отчеты за неделю, месяц, квартал, год
  - Визуализация данных (круговые и столбчатые диаграммы)
  - Сравнение с предыдущими периодами
  - Экспорт данных в CSV

- **🎯 Управление бюджетом**
  - Установка лимитов по категориям
  - Отслеживание расходов
  - Уведомления при приближении к лимиту
  - Прогнозирование расходов

- **📋 Категории**
  - 8 предустановленных категорий
  - Создание своих категорий с выбором иконки
  - Редактирование названия и иконки
  - Удаление категорий (мягкое, расходы сохраняются)
  - Иконки и цвета для категорий

- **💱 Валюты**
  - Поддержка 6 валют: RUB, USD, EUR, GBP, CNY, KZT
  - Автоматическое обновление курсов
  - Конвертация при добавлении расходов

- **🔔 Напоминания**
  - Ежедневное напоминание в 21:00 (MSK)
  - Настраиваемое время отправки
  - Автоматическая рассылка всем пользователям

## 🚀 Установка

### Требования

- Python 3.10 или выше
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather))
- Exchange Rate API Key (опционально, для актуальных курсов валют)

### Быстрый старт

1. **Клонирование репозитория**
```bash
git clone <repository-url>
cd finance-telegram-bot
```

2. **Создание виртуального окружения**
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate
```

3. **Установка зависимостей**
```bash
pip install -r requirements.txt
```

4. **Настройка переменных окружения**
```bash
# Windows
copy .env.example .env

# Linux/macOS
cp .env.example .env
```

Отредактируйте файл `.env` и добавьте ваш токен бота:
```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
EXCHANGE_API_KEY=your_api_key_here  # опционально
```

5. **Запуск бота**
```bash
python -m bot.main
```

Или используйте скрипт запуска:
```bash
# Windows
run.bat

# Linux/macOS
chmod +x run.sh
./run.sh
```

## 📝 Использование

### Быстрое добавление расходов (с автоматическим определением категории!)

Просто отправьте сообщение боту, и категория определится автоматически по ключевым словам:

```
500 транспорт поездка на такси    → 500₽, категория "Транспорт", описание "поездка на такси"
1500 еда обед в ресторане         → 1500₽, категория "Еда и напитки", описание "обед в ресторане"
3000 аптека лекарства             → 3000₽, категория "Здоровье", описание "лекарства"
100$ одежда новая куртка          → 100$, категория "Одежда", описание "новая куртка"
2500 кино билеты                  → 2500₽, категория "Развлечения", описание "билеты"
```

**Поддерживаемые ключевые слова для категорий:**
- 🍔 **Еда и напитки**: еда, обед, ужин, завтрак, кафе, ресторан, кофе, продукты, супермаркет
- 🚗 **Транспорт**: транспорт, такси, метро, автобус, бензин, заправка, парковка, uber
- 🏠 **Жилье**: жилье, аренда, квартплата, коммуналка, ипотека, жкх, интернет, ремонт
- 💊 **Здоровье**: здоровье, аптека, лекарства, врач, больница, фитнес, спорт, зал
- 👕 **Одежда**: одежда, обувь, куртка, штаны, кроссовки, магазин, shopping
- 🎮 **Развлечения**: развлечения, кино, театр, концерт, клуб, бар, игры, netflix
- 📚 **Образование**: образование, курсы, обучение, книга, учеба, университет

Если категория не распознана, расход добавится в категорию "Прочее".

### Добавление расхода с выбором категории

1. Отправьте команду `/add` или нажмите кнопку "💰 Добавить расход"
2. Выберите категорию
3. Введите сумму и описание

### Получение отчета

1. Отправьте команду `/report` или нажмите "📊 Отчеты"
2. Выберите период (неделя, месяц, квартал, год)
3. Получите текстовый отчет и графики

### Изменение категории расхода

Если вы добавили расход в неправильную категорию:

1. Отправьте `/recent` для просмотра последних расходов
2. Используйте команду `/move_ID` (например, `/move_42`)
3. Выберите новую категорию
4. Готово! Расход перемещён

Или сразу после добавления расхода нажмите кнопку **"✏️ Изменить категорию"**

### Установка бюджета

1. Отправьте команду `/budget`
2. Выберите "➕ Установить бюджет"
3. Выберите категорию — отобразится текущий лимит
4. Введите сумму лимита

### Управление категориями

1. Отправьте команду `/categories`
2. **Создание**: нажмите "➕ Добавить категорию" → введите название → выберите иконку
3. **Просмотр**: нажмите "📋 Мои категории" — список с кнопками ✏️ и 🗑️
4. **Редактирование**: нажмите ✏️ у нужной категории → введите новое название → выберите иконку
5. **Удаление**: нажмите 🗑️ у категории → подтвердите удаление

### Удаление расхода

После добавления расхода появится кнопка **"🗑️ Удалить расход"**. Нажмите её и подтвердите удаление.

LOG_LEVEL=INFO
DEFAULT_CURRENCY=RUB
CURRENCY_UPDATE_INTERVAL=6

# Прокси (если Telegram API заблокирован)

## 🐛 Отладка

### Логи

Логи сохраняются в файл `logs/bot.log` и выводятся в консоль.

Уровни логирования:
- `DEBUG` - детальная отладка
- `INFO` - основные события
- `WARNING` - предупреждения
- `ERROR` - ошибки

### Резервное копирование

База данных хранится в файле `data/finance_bot.db`. Рекомендуется регулярно создавать резервные копии:

```bash
# Linux/macOS
cp data/finance_bot.db data/backups/finance_bot_$(date +%Y%m%d).db

# Windows
copy data\finance_bot.db data\backups\finance_bot_%date%.db
```
