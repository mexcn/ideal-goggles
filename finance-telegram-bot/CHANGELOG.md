# Changelog - Улучшения бота

## Версия 1.1.0 - Улучшенный парсинг и управление

### ✨ Новые возможности

#### 1. Автоматическое определение категорий
- **Умный парсинг текста**: Теперь бот автоматически определяет категорию по ключевым словам
- **Формат**: `500 транспорт поездка на такси` → автоматически определяется категория "Транспорт"
- **Поддержка русских ключевых слов** для всех 7 основных категорий

**Примеры:**
```
1500 еда обед в ресторане        → Категория: Еда и напитки
500 такси до работы              → Категория: Транспорт
3000 аптека витамины             → Категория: Здоровье
2500 кино билеты                 → Категория: Развлечения
```

**Ключевые слова по категориям:**
- 🍔 **Еда и напитки**: еда, обед, ужин, завтрак, кафе, ресторан, кофе, продукты, супермаркет, магазин
- 🚗 **Транспорт**: транспорт, такси, метро, автобус, трамвай, бензин, заправка, парковка, uber, яндекс
- 🏠 **Жилье**: жилье, аренда, квартплата, коммуналка, ипотека, жкх, электричество, интернет, ремонт
- 💊 **Здоровье**: здоровье, аптека, лекарства, врач, больница, клиника, таблетки, фитнес, спорт, зал
- 👕 **Одежда**: одежда, обувь, куртка, штаны, рубашка, платье, кроссовки, магазин, shopping
- 🎮 **Развлечения**: развлечения, кино, театр, концерт, клуб, бар, игры, netflix, spotify, отдых
- 📚 **Образование**: образование, курсы, обучение, книга, учеба, университет, репетитор, тренинг

#### 2. Перемещение расходов между категориями
- **Новая команда** `/recent` - просмотр последних 10 расходов
- **Команда перемещения** `/move_ID` - изменить категорию конкретного расхода
- **Inline кнопка** "✏️ Изменить категорию" сразу после добавления расхода

**Как использовать:**
1. Добавьте расход (случайно выбрали не ту категорию)
2. Нажмите кнопку "✏️ Изменить категорию"
3. Выберите правильную категорию
4. Готово!

Или:
1. Отправьте `/recent` для просмотра последних расходов
2. Используйте `/move_42` (где 42 - ID расхода)
3. Выберите новую категорию

#### 3. Удалены Reply-клавиатуры
- **Убраны** навязчивые кнопки внизу экрана (Категории, Отчеты, Итого, Помощь)
- **Добавлено** встроенное меню команд через BotFather
- **Доступ к командам** через кнопку меню (/) в Telegram

**Список команд в меню:**
- `/start` - Запустить бота
- `/help` - Помощь и инструкции
- `/add` - Добавить расход
- `/recent` - Последние расходы
- `/report` - Отчеты и графики
- `/budget` - Управление бюджетом
- `/categories` - Управление категориями
- `/stats` - Быстрая статистика
- `/settings` - Настройки

### 🔧 Технические улучшения

#### Обновленный парсер (`bot/utils/parsers.py`)
- Новая функция `_extract_category_from_text()` для извлечения категории
- Расширенный словарь ключевых слов для каждой категории
- Улучшенная обработка текста с удалением ключевых слов из описания
- Возврат кортежа `(category_name, description)` вместо только описания

#### Улучшенный handler расходов (`bot/handlers/expenses.py`)
- Функция `quick_expense()` теперь использует автоопределение категории
- Новые handlers:
  - `recent_expenses_command()` - просмотр последних расходов
  - `move_expense_start()` - начало перемещения расхода
  - `move_expense_to_category()` - завершение перемещения
  - `move_expense_command()` - команда /move_ID
- Inline кнопка для быстрого редактирования после добавления

#### Настройка команд (`bot/main.py`)
- Автоматическая установка команд через `bot.set_my_commands()`
- Команды отображаются в меню Telegram при нажатии на "/"

```

Бот: ✅ Расход добавлен!
     💰 500.00 ₽
     🍔 Еда и напитки

     [✏️ Изменить категорию]
```

```
Бот: 📋 Последние расходы:
     🍔 500.00 ₽ - обед в кафе

Пользователь: /move_123
Бот: [Показывает категории для выбора]

Пользователь: [Выбирает 🚗 Транспорт]
Бот: ✅ Расход перемещён!
     Новая категория: 🚗 Транспорт
```
